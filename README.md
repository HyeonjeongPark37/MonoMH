# MonoMH: Modeling and Learning Multiple Hypotheses for Monocular 3D Object Detection (WACV 2026)

[![paper](https://img.shields.io/badge/WACV-2026-blue.svg)](#)
[![license](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![python](https://img.shields.io/badge/Python-3.9+-informational.svg)](#)
[![pytorch](https://img.shields.io/badge/PyTorch-2.x-orange.svg)](#)

**MonoMH** is a monocular 3D object detector that models **multiple plausible 3D hypotheses** per object and learns to select them for robust 3D localization.
